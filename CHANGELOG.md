# Changelog

## Unreleased

### Changed
- 清理历史遗留代码：移除未使用的 sparkline 模块及其测试。
- 清理未使用函数：移除 `fixed_speed_token` 及对应测试。
- 图表渲染模块只保留运行时使用的单图渲染函数，删除未使用的合并图渲染函数与测试。
- 文档与发布命令同步到当前打包入口（`IP Tray.spec` + `launcher.py`）。

## 0.3.0 - 2026-02-22

### Added
- 菜单新增图形化网速历史显示：5 分钟时间窗、接收/发送分离图表。
- 历史曲线支持彩色渲染（接收蓝色、发送橙色），并增强为接近系统面板的视觉样式。
- 新增图像渲染测试，覆盖历史图尺寸与输出有效性。

### Changed
- 托盘显示改为仅显示国家旗帜，网速详情集中到菜单中。
- 菜单网速改为活动监视器风格（接收/发送实时速率 + 对应历史图）。
- 修复菜单展开时定时刷新受 runloop mode 影响的问题，保证展开状态下数据持续更新。
- 修复 `python -m ip_tray.app` 可能出现的 `runpy` 预加载告警（包入口改为惰性导入）。
- 修复 PyInstaller 打包后启动失败问题（入口改为 `launcher.py`，与 `IP Tray.spec` 对齐）。

## 0.2.0 - 2026-02-22

### Added
- 菜单内运行时配置：刷新频率、公网刷新间隔、请求超时、通知开关。
- 配置持久化：设置自动保存并在重启后恢复。
- 单元测试：新增网络格式化、回退逻辑、配置读写与校验测试。
- CI 工作流：自动安装依赖、运行单元测试、执行离线开发检查。
- 发布检查清单文档：`RELEASE_CHECKLIST.md`。

### Changed
- 定时刷新循环不再静默吞错，改为分层异常处理并提供状态显示。
- 开发检查脚本支持 `--skip-network`，适配 CI/离线环境。
- 项目元数据更新：版本提升至 `0.2.0`，补充 `readme`、`license`、`classifiers` 等字段。
